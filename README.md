# Ecuador Stock Market Dashboard (Equities)

Interactive Streamlit dashboard to explore Equities traded on Ecuador’s exchanges (BVQ and BVG) using data hosted on Hugging Face Hub (HF Hub).

**Acronyms:**
- **KPI (Key Performance Indicator)**: metric used to evaluate activity or performance.
- **API (Application Programming Interface)**: a way for systems to communicate.
- **HF Hub (Hugging Face Hub)**: collaborative repository of datasets and models.

## Features

- Filters bydate range, issuer, and exchange (BVQ/BVG).
- KPIs: Traded Amount, Traded Volume, and Record Count.
- Time series with selectable frequency: Day / Week / Month.
- Market share (donut chart) by issuer with Top-N + “Others”.
- Exchange share (BVQ vs. BVG).
- Detailed table with CSV export.
- “Methodology” panel with definitions and caveats.
